What the model predicts
Binary outcome — 1 (Success) if a startup achieves Revenue > ₹1 Cr or Valuation > ₹30 Cr within 18 months of incubation. Otherwise 0 (Needs support). These thresholds are adjustable at programme level.

Importances computed via Random Forest gain on the full 198-row SAMRIDH dataset. Team size dominates because cohort-level operational maturity strongly correlates with revenue outcomes in this dataset.
Algorithm — XGBoost gradient boosting
XGBoost builds 300 decision trees sequentially. Each tree learns from the errors of the previous one (gradient boosting). All trees vote together to produce a final probability.

Hyperparameters used: max_depth=4, learning_rate=0.07, subsample=0.8, colsample_bytree=0.8, n_estimators=300

Validation: Stratified 5-fold cross-validation (essential for a 198-row dataset). Evaluated on AUC-ROC and F1 score — not raw accuracy, which is misleading on imbalanced classes.

Retraining cadence
Each new SAMRIDH cohort adds ~30–50 rows. Retrain the model with the updated dataset and compare feature importance drift across cohorts. This turns the programme into a self-improving selection engine over time.

Run python samridh_xgboost.py with the updated Excel file after each cohort.
